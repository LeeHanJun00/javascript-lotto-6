# 로또 - 구현 기능 목록

## 📢 요구 사항
<프로그래밍 요구사항>
- depth를 3이 넘지 않도록 구현
- else를 지양한다.

<당첨번호>
- 서로다른 번호
- 숫자범위 : 1 ~ 45
- 입력시 쉼표기준으로 구분
- 번호 6개 + 보너스번호 1개

<추첨번호>
- 1 ~ 45 서로다른 숫자
- 6 자리

<로또>
- 로또 1장당 1,000원
- 1,000원으로 나누어 떨어지게 입력 받기

<수익률>
- 소수점 둘째 자리 반올림
- (당첨금액/구임금액)*100

## 📍 프로그램 흐름
- 시작
1. 구입 금액 입력
2. 예외처리
3. 구매한 로또 갯수 출력
4. 갯수만큼 추첨번호 출력
5. 당첨번호 입력 
6. 예외처리
7. 보너스번호 입력 
8. 예외처리
9. 당첨확인 기능 
10. 당첨내역 출력 
11. 수익률 계산기능 
12. 수익률 출력
- 종료
